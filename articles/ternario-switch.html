Ternário e Switch em Javascript

JAVASCRIPT

Operador Ternário

O operador ternário é uma estrutura simplificada de um bloco if/else com o mesmo resultado, porém sintaxe menor. A sintaxe do operador ternário é (condição) ? valor_se_verdadeiro : valor_se_falso. É como se a própria sintaxe perguntasse "a condição é verdadeira?" e a resposta é sim ou não, em que o operador : atua como o "ou". Se a condição retorna true, o primeiro valor é executado, se retorna false, o segundo valor é executado.

Este operador é bastante utilizado quando é necessário condicionar o valor de uma variável, para realizar renderizações condicionais ou apenas deixar o código mais conciso.

É possível aninhar operadores ternários, tendo como valor_se_falso uma nova estrutura de ternário com uma nova condição e também é possível realizar mais de uma operação em cada bloco de true ou false, separando as operações por vírgulas como em (condição) ? operação1, operação2 : operação3, operação4.

Switch

O switch é uma estrutura de condicional muito utilizada para evitar muitas utilizações suscessivas de blocos if/else if/else, deixando o código mais conciso. Ele é utilizado quando uma expressão necessita ter seu valor avalido e possui resultados diferentes de acordo com seu valor.

A sua estrutura básica é:

switch (condição) {
  case "opção1":
  //bloco de código a ser executado caso a opção1 seja verdadeira
  break;
  case "opção2":
  //bloco de código a ser executado caso a opção2 seja verdadeira
  break;
  ...
  default:
  //bloco de código a ser executado caso nenhuma das opções seja verdadeira.
  break;
}

Esta estrutura possui algumas palavras reservadas como o switch, que inicia a estrutura; o case que indica um possível valor da condição; o break, que faz com que demais opções não sejam testadas após o programa encontrar a verdadeira e o default que é um case particular em que nenhum dos cases testados retornaram verdadeiros, atuando como se fosse um else.
