Objetos em Javascript

JAVASCRIPT

Em Javascript, tudo pode ser considerado um objeto. Para efeitos de definição, um objeto é tudo aquilo que possui propriedades, sejam estas propriedades pares de chave/valor ou funções (que, no caso de serem uma propriedade de um objeto, são chamadas métodos). As propriedades são as características do objeto, podendo ser adicionadas, acessadas, removidas e modificadas. Nesta linguagem de programação, todos os objetos são descendentes do objeto Object. Desta forma, é possível utilizar os métodos de Object em todos os objetos criados por conta da herança destes métodos.

Criação de um objeto

Um objeto é usualmente criado de duas formas: pela forma literal ou utilizando uma função construtora. Na forma literal, o objeto é definido como uma variável e suas os pares chave/valor das suas propriedades são escritas dentro de chaves. Utilizando uma função construtora, primeiramente é criada uma função com o "esqueleto" do objeto e, então, um objeto do tipo da função construtora é instanciado com a palava reservada new.

A palavra reservada this na função construtora se refere ao próprio objeto. Em uma função com o nome Objeto propriedades nome e idade, this.nome seria a mesma coisa que Objeto.nome. O uso desta palavra reservada é indicado para minimizar a repetição do nome do objeto ao longo de uma função construtora ou em métodos de objetos que utilizem suas propriedades.

Acessar propriedade do objeto

Para acessar as propriedades de um objeto, existem duas notações principais: a notação de ponto e a notação de array. As notações sempre iniciam com o nome do objeto e indicam a propriedade a ser acessada de acordo com sua sintaxe. Para a notação de ponto temos objeto.propriedade e para a notação de array, temos objeto["propriedade"]. Tomando como base um objeto chamado obj com propriedades nome, idade e cargo, podemos acessar os valores destas propriedades das seguintes formas: obj.nome, obj.idade e obj.cargo ou obj["nome"], obj["idade"] e obj["cargo"]. Caso exista uma variável contendo como seu valor o nome de uma das propriedades do objeto, também é possível, por meio da notação de array, acessar a propriedade por meio de objeto[variável].

Adicionar propriedade do objeto

Para adicionar uma propriedade ao objeto, dado que ela ainda não exista, é possível utilizar dois formatos. Um formato mais simples seria "atribuir" um valor à uma propriedade não existente, como se ela fosse uma variável. Para um objeto cujo nome é obj, poderíamos criar uma propriedade com obj.novaPropriedade = valor. Desta forma, a novaPropriedade já seria adicionada ao objeto, dado que ele exista. Uma forma mais completa é a função Object.defineProperty. Esta função possui 3 argumentos a serem indicados, o primeiro sendo o objeto no qual a propriedade será adicionada, o segundo é o nome da propriedade e o terceiro é o descritor da propriedade. Neste descritor, pode-se adicionar, além do valor da propriedade a ser adicionada, métodos get e set, se a propriedade pode ser alterada ou não, entre outras "configurações". Remover propriedade do objeto

Para remover uma propriedade de um objeto, dado que ela exista, é necessário apenas utilizar a palavra reservada delete e então referenciar a propriedade que se deseja deletar por meio das notações de ponto ou array. Exemplos genéricos disto são delete objeto.propriedade e delete objeto["propriedade"].

Mudar valor de propriedade do objeto

Para mudar o valor de uma propriedade de um objeto, dado que ela exista, é necessário apenas reatribuir um valor à ela, como se a propriedade fosse uma variável. Na notação de ponto, temos objeto.propriedade = novo_valor e, na notação de array, temos objeto["propriedade"] = novo_valor.

Comparar objetos

Na linguagem Javascript, objetos são um tipo de referência. Deste modo, todo objeto é igual apenas a si próprio. Comparar dois objetos diferentes, mesmo que eles possuam o mesmo conteúdo, retornará false pois a linguagem interpretará como duas referências diferentes, apesar do conteúdo igual.
