Variáveis CSS

CSS BÁSICO

As variáveis CSS são valores fixos criados pelo desenvolvedor para propriedades que se repetem bastante ao longo do código. Podem representar cores, dimensões, fontes, entre outros tipos de valores. As variáveis são facilitadoras de manutenção de código, visto que, para variáveis global, cada mudança só precisa ser alterada em um local, o da inicialização da variável.

A inicialização de uma variável CSS pode se dar de 3 formas: por meio do pseudo elemento :root e pela regra @property, gerando variáveis globais, ou apenas pela definição de variáveis e valor em um bloco de código com -- no início do nome, gerando variáveis locais.

A grande diferença entre as formas de inicialização de variáveis, além de seu escopo, é que a regra @property confere maior controle sobre o tipo da variável e se ela herda valores de elementos pais.

Na utilização, a variável deve ser englobada pelo var(), ou seja, para a utilização de uma cor definida como --cor-principal: #ffffff temos, como exemplo, color: var(--cor-principal). A utilização seguirá esta regra independente do modo de inicialização da variável.

É possível, também, definir valores de backup para variáveis, para caso ela venha a receber um valor invalido. Por exemplo, se uma variável --cor-principal for inicializada com o valor 16px, ela gerará um valor inválido caso utilizada em propriedades de cor. Caso a propriedade tenha seu valor indicado como var(--cor-principal, #ffffff), ela assumirá o valor do segundo argumento, que é válido para propriedades de cor. Caso nenhum argumento dentro dos parênteses do var() sejam válidos, a propriedade retornará seu valor inicial default ou um valor herdado por um elemento pai com a propriedade definida, se existir.

Variáveis globalmente inicializadas podem ser sobrescritas localmente, porém o escopo desta sobrescrição será somente este: local. Utilizações da variável em outros blocos de código seguirão retornando o valor definido globalmente.
