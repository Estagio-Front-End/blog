Fontes personalizadas

CSS BÁSICO

Nem sempre as fontes padrão do sistema operacional irão preencher os requisitos do layout, de modo que é possível inserir novas fontes por meio do CSS. As fontes podem ser importadas por meio de link no documento HTML e @import ou @font-face no documento CSS. Nos dois primeiros casos, sites como o Google Fonts já facilitam o processo ao ter o código a ser inserido já disponibilizado à medida que você escolhe as fontes que deseja. No último caso, as fontes são baixadas em formatos como .ttf ou .woff e referenciadas no código.

Notar que, se uma fonte baixada possui vários pesos, ou seja, vários arquivos dentro de uma mesma família de fontes, é necessário que se faça um @font-face para cada peso.
