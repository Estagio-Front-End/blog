Box model e medidas em CSS

CSS BÁSICO

Box model

O box model é o modelo pelo qual elementos HTML possuem sua altura e largura total definidas, assemelhando-se a uma caixa. Ele é composto de 4 elementos, o content, que é o conteúdo do elemento; o padding, que é uma área ao redor do conteúdo; a border, que é uma borda que circunda o conteúdo e a margin, que é uma área exterior à borda.

A propriedade box-sizing pode definir como a altura e largura de elementos são calculadas. O valor default é content-box, em que espessuras de padding e border não são consideradas no cálculo de dimensões, apenas o conteúdo. Já com o valor border-box, as espessuras de padding e são consideradas no cálculo, além das dimensões do próprio conteúdo. Outros valores possíveis para esta propriedade são initial, que retorna ao valor default e inherit, em que o elemento herda o valor de box-sizing do elemento pai.

Medidas em CSS

As medidas em CSS possuem valores absolutos e relativos. Em geral, valores absolutos são recomendados para casos em que o meio final é fixo e conhecido, visto que valores absolutos podem ser apresentados de modo diferente em cada tamanho de tela disponível.

Os valores absolutos de medidas podem ser expressos em cm, mm, in, px, pt e pc. A medida absoluta em px é a mais utilizada.

Medidas relativas são mais comumente utilizadas para garantir a adaptabilidade das medidas às diferentes tipos de meios finais (mobile, desktop, tablets, entre outros). As medidas relativas mais utilizadas são:

  • em: esta medida é relativa ao tamanho da fonte sendo utilizada no elementos. Caso não seja alterada, por default, 1em equivale à 16px.
  • rem:esta medida é relativa ao tamanho de fonte definido no elemento :root na folha de estilo. Um artifício bastante utilizado para evitar conversões dificultosas entre em e px, é de adicionar font-size: 62.5% ao elemento root, de modo que 1rem será equivalente à 10px. Desta forma, 1.7rem equivale à 17px, o que é mais facilmente lembrado e escrito do que 1.0625em.
  • vw: esta medida é relativa à largura do viewport (largura da tela do dispositivo), em que 1vw equivale a 1% da largura do viewport. Amplamente utilizada para garantir a responsividade de aplicações web.
  • vh: esta medida é relativa à altura do viewport (altura da tela do dispositivo), em que 1vh equivale a 1% da alltura do viewport. Amplamente utilizada para garantir a responsividade de aplicações web.
  • %: esta medida é relativa ao tamanho do elemento pai. Desta forma, se temos um elemento pai com largura de 500px, e um elemento filho com largura de 50%, a largura absoluta do elemento filho será 250px.

Além das unidades supracitadas, também existem ex, ch, vmin e vmax.
