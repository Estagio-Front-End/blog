Propriedades em Javascript

JAVASCRIPT

  • elemento.outerHTML: esta propriedade possibilita o acesso à estrutura HTML do elemento selecionado e, caso o elemento tenha filhos, os retorna também.
  • elemento.innerHTML: esta propriedade possibilita o acesso aos elementos HTML dentro de um elemento pai ou a modificação destes elementos, seja sobrescrevendo-os ou adicionando novos elementos filhos.
  • elemento.innerText: esta propriedade possibilita o acesso ao conteúdo textual de um elemento. Possui quase a mesma função de textContent, com a diferença sendo em casos de elementos com visibilidade ocultada pelo CSS. No textContent, a mudança de visibilidade não possui influência e, caso seja atribuído à uma variável, ela irá mostrar o texto. No innerText, a mudança de visibilidade é totalmente aplicada, de modo que, caso atribuída a uma variável, esta irá retornar uma string vazia.
