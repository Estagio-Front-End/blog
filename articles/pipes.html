Pipes em Angular

PREVIEW AQUI

Por Camila Matias | Desenvolvedora Front End

Publicado: 03 Junho 2024

  2. Pipes do Angular

Pipes, no Angular, são funções capazes de transformar dados de acordo com o que se deseja, seja uma data, uma frase, um valor monetário, entre outros, dentro do template de um componente. O Angular possui vários pipes já embutidos, porém também dá ao desenvolvedor a possibilidade de criar seus próprios pipes.

Para utilizar os pipes, basta importar e adicionar os nomes dos pipes que se deseja utilizar no imports do componente (versões acima da 16) ou do módulo associado ao componente em que se deseja utilizar os pipes.

Como os pipes são do módulo common do Angular, também é possível importar apenas o CommonModule e ter acesso à todos os pipes implicitamente. A documentação da API do Angular possui, na seção common, todos os pipes, suas descrições e explicações de uso.
