Pipes em Angular

PREVIEW AQUI

Por Camila Matias | Desenvolvedora Front End

Publicado: 03 Junho 2024

Navegue pelo conteúdo

  1. O que são pipes
  2. Pipes do Angular
  3. Pipes customizados

Pipes, no Angular, são funções capazes de transformar dados de acordo com o que se deseja, seja uma data, uma frase, um valor monetário, entre outros, dentro do template de um componente. O Angular possui vários pipes já embutidos, porém também dá ao desenvolvedor a possibilidade de criar seus próprios pipes.

Para utilizar os pipes, basta importar e adicionar os nomes dos pipes que se deseja utilizar no imports do componente (versões acima da 16) ou do módulo associado ao componente em que se deseja utilizar os pipes.

Como os pipes são do módulo common do Angular, também é possível importar apenas o CommonModule e ter acesso à todos os pipes implicitamente. A documentação da API do Angular possui, na seção common, todos os pipes, suas descrições e explicações de uso.

Pipes do Angular

Como mencionado anteriormente, o Angular possui alguns pipes embutidos em sua API. Para a utilização de um pipe dentro de um template, é necessário, após o valor que se deseja transformar, incluir o caractere |, chamado de pipe, o tipo de pipe que se deseja utilizar e, se necessário, os argumentos cabíveis. Os mais comumente utilizados são:

  • DatePipe: transforma datas para variados formatos contendo manipulação de dia, mês, ano, hora, dia da semana, entre outros. Para utilização, se inclui | date : argumentos (ex: "dd/MM/yyyy").
  • UpperCasePipe: transforma um texto para que esteja com todas as letras em maiúsculo. Para utilização, se inclui | uppercase.
  • LowerCasePipe: transforma um texto para que esteja com todas as letras em minúsculo. Para utilização, se inclui | lowercase.
  • CurrencyPipe: transforma um número em uma string com a indicação monetária. Para utilização, se inclui | currency : argumentos (ex: "BRL" para real brasileiro).
  • DecimalPipe: transforma um número com casas decimais de acordo com a regra explicitada. Para utilização, se inclui | number : regra (ex: "2.0-2" para 2 números antes da vírgula e até 2 casas decimais)
  • PercentPipe: transforma um número para seu equivalente em porcentagem. Para utilização, se inclui | percent.
  • SlicePipe: transforma um array trazendo um recorte do array original. Para utilização, se inclui | slice : início:fim (ex: 2:6 para trazer do elemento de índice 2 ao elemento de índice 5).
  • JsonPipe: torna os dados no formato JSON visíveis, em vez de retornar o object Object. Para utilização, se inclui | json.
  • AsyncPipe: inscreve automaticamente um Observable ou uma Promise, de modo que é muito utilizado para detectar de forma automática mudanças em dados assíncronos. Para utilização, se inclui | async.

É possível encadear pipes, porém é necessário ter cuidado com a ordem de transformações para que não se realize transformações indesejadas ou crie ordens que não são possíveis de serem executadas.

Pipes customizados

O Angular também permite que sejam criados pipes customizados para que o próprio desenvolvedor defina como será a transformação do dado que passará pelo pipe. Para criar um pipe, basta utilizar o comando ng generate pipe nome-pipe, em que nome-pipe é um nome arbitrário, ou utilizar o seu shorthand ng g p nome-pipe. Este comando criará 2 arquivos, um nome-pipe.pipe.ts, que conterá a lógica do pipe criado e o nome-pipe.pipe.spec.ts, que conterá os testes unitários acerca da lógica do pipe criado.

A estrutura inicial de um arquivo de pipe contém os imports necessários, o decorator @Pipe, onde estará a nomenclatura do pipe e a classe, que conterá toda a lógica de transformação de dados. Esta transformação se dá pelo método transform, que recebe o valor a ser transformado e quaisquer argumentos necessários para a transformação, e retorna o dado modificado.

Para a utilização do pipe customizado em um template, basta importar sua classe no componente em que se quer utilizá-lo e inserí-lo no template como qualquer outro pipe, utilizando o nome que foi dado no decorator @Pipe.
