Transitions

ANIMAÇÕES

As transitions, como o nome já denota, são transições de estado de elementos, se assemelhando à animações. Muito utilizadas em conjunto com as pseudo-classes como :hover, :active e :focus, são, em geral, utilizadas para que haja uma animação quando o usuário clica ou passa o cursor em algum item da página.

Além de mudança de propriedades já definidas, com a propriedade transform, é possível mover, rotacionar, distorcer, aumentar ou diminuir o tamanho de elementos.

Para aplicar a transition à um elemento, após definir todas as modificações a serem realizadas por meio de uma pseudo-classe, é necessário adicionar as propriedades de transition no bloco de estilização do elemento original. Estas propriedades são:

  • transition-property: propriedade que sofrerá uma alteração, como margin, background-color, entre outras.
  • transition-duration: tempo no qual a alteração ocorrerá.
  • transition-timing-function: permite estabelecer uma curva de aceleração da transição. Recebe valores como steps(), ease-in, linear, a depender de como se queira que a transição ocorra.
  • transition-delay: quanto tempo demorará para a transição iniciar
  • transition-behavior: define se animações do tipo discreta (que possuem apenas um valor para início e um valor para fim) serão permitidas.

Para diminuição do código, existe a shorthand transition que engloba todas estas propriedades um uma só linha. A aplicação das transitions pode ser única, caso todas as transitions aplicadas tenham os mesmos valores de propriedades, ou pode ser customizada de acordo com a necessidade.
