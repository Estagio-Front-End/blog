Eventos em Javascript

JAVASCRIPT

Eventos são, como o próprio nome diz, eventos que ocorrem, geralmente decorrentes de uma interação do usuário com a aplicação. É possível que se crie blocos de código designados para tratar estes eventos e dispará-los quando o evento ocorrer, gerando uma resposta imediata à ação do usuário. Estes eventos podem ser vários, como o clique em um botão, o envio de um formulário, a finalização do carregamento de uma página, entre outros.

Para lidar com um evento, em geral, temos 3 blocos de código: a seleção do elemento no qual o evento irá ocorrer, a adição de um "escutador" de eventos ao elemento e a criação da função que irá lidar com este evento. Estes 3 blocos são representados pela seleção do elemento com métodos como querySelector ou getElementById, o acoplamento do método addEventListener ao elemento e a criação de uma função de callback. Uma função de callback é uma função que é passada como argumento de uma outra função. Muito utilizada em funções assíncronas antes das Promises, atualmente uma grande utilização é no tratamento de eventos.

Caso se queira cancelar o comportamento padrão de um evento, utiliza-se o método preventDefault(). Para que este método funcione da forma como deveria, a propriedade cancelable do evento deve ser true, fazendo com que o evento seja cancelável.
