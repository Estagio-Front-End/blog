Javascript

JAVASCRIPT

O Javascript é uma linguagem de programação interpretada que, junto ao HTML e CSS, integra a tríade básica do desenvolvimento web. O HTML é utilizado para a marcação da página, o CSS para sua estilização e, o Javascript, para tornar a página interativa. Desta forma, é possível que ações do usuário, como o clique de um botão ou o scroll pelo conteúdo, reflitam em alguma modificação na página, como alterações no próprio HTML ou CSS, alertas, entre outras.

Por ser muito popular, o Javascript possui muitas funcionalidades desenvolvidas para sua melhoria como APIs, bibliotecas e frameworks. Estas funcionalidades tornam o desenvolvimento em Javascript mais robusto e acelerado, sendo amplamente utilizadas.

Para que um código Javascript seja aplicado em uma página, é necessário utilizar a tag script no documento HTML, podendo esta utilização ser de duas formas e em duas localidades diferentes.

Em termos de forma, podemos ter o Javascript dentro do próprio documento, assim como um CSS interno, escrevendo todo o código dentro de uma tag script. Podemos, também, ter um arquivo com extensão .js à parte do documento HTML, que será linkado ao documento por meio do atributo src da tag script. A principal diferença entre estas duas formas é que, tendo o script interno, ele só será aplicado àquela página em específico, ao passo que com o script externo, é possível linkar esse arquivo de tal forma que um só bloco de código atue em várias páginas diferentes.

Em relação à localização da tag script, ela poderá estar logo após a tag head, ou logo antes do fechamento da tag body. A principal diferença entre essas localizações é que, devido ao documento HTML ser lido na ordem em que é escrito, caso o script necessite que o HTML esteja totalmente carregado para operar e esteja escrito antes do body, problemas ocorrerão e o script não atuará. Desta forma, se o script necessitar do HTML carregado, ele deve estar antes do fechamento da tag body, caso contrário, ele pode ser linkado ou escrito após a tag head.

Para programar em Javascript, é necessário conhecer sua sintaxe básica como declaração de variáveis, os tipos de dados e principais operadores.

Variáveis

Existem 3 palavras reservadas para a declaração de variáveis em Javascript: var, let e const. Clique em uma das opções para mostrar a explicação sobre declaração, atribuição e alteração de cada um dos tipos de variáveis.

É importante ressaltar que a principal diferença entre variáveis declaradas com var e let é seu escopo. Variáveis declaradas com var terão escopo global, ou seja, ao longo do código, uma variável declarada com var permanecerá com seu valor, exceto se reatribuída. Variáveis declaradas com let possuem escopo local, de modo que, se uma variável foi declarada dentro de uma função, condicional ou loop, fora desses blocos de código ela não existirá e seu valor não poderá ser acessado, exceto em casos em que uma função retorne o seu valor.

Tipos de dados

Em Javascript, temos alguns tipos de dados como number, string, object, boolean, array, null e undefined. Escolha um tipo na caixa de seleção para ter mais informações e exemplos de cada tipo.

Operadores

Os operadores são símbolos utilizados para realizar operações entre números, strings ou números e strings. Clique em cada botão de "mostrar" para visualizar mais informações de cada operador.

Operador de soma (+)
O operador + realiza operações de soma entre números, um operando incrementado a outro. Em operações entre números e strings, strings contendo números e strings contendo texto os operandos são concatenados. Desta forma, uma operação de soma entre "5" e 5 terá 55 (e tipo string) como resultado sendo string somente o primeiro operando, somente o segundo operando ou os dois operandos. Da mesma forma, uma operação de soma entre "Palavra" e "Chave" terá como resultado "PalavraChave".
Operador de subtração (-)
O operador - realiza operações de subtração entre números diminuindo um operando do outro. Em operações entre números e strings contendo números ou entre duas strings contendo números, os operandos serão "convertidos" em dados do tipo number, gerando um resultado da operação do tipo number. Subtrações entre strings contendo texto gerarão um resultado "NaN" que significa "Not a Number", em português: não é um número.
Operador de multiplicação (*)
O operador * realiza operações de multiplicação entre números, um operando multiplicado pelo outro, em que a ordem dos fatores não alterará o resultado. Tentativas de multiplicação entre números e strings ou entre duas strings contendo texto gerarão um resultado "NaN" que significa "Not a Number", em português: não é um número. Em multiplicações entre duas string contendo números, o código é "convertido" em número e a operação será realizada, gerando um número com o resultado da multiplicação.
Operador de divisão (/)
O operador / realiza operações de divisão entre números, ou seja, o resultado da divisão de um operando por outro operando. Importante lembrar que, como na matemática, divisões por zero não são permitidas em código e gerarão erros. Tentativas de divisão entre números e strings ou entre strings contendo texto gerarão um resultado "NaN" que significa "Not a Number", em português: não é um número. Em divisões entre duas string contendo números, o código é "convertido" em número e a operação será realizada, gerando um número com o resultado da divisão.
Operador de resto de divisão (%)
O operador % realiza operações de resto da divisão entre números, ou seja, dividindo um operando pelo outro, qual será o resto desta divisão. Importante lembrar que, como na matemática, divisões por zero não são permitidas em código e gerarão erros. Este operador é muito utilizado em validações de números pares/ímpares.
Operador de potência (**)
O operador ** realiza operações de potência entre números, ou seja, um operando elevado à potência de um segundo operando.

Existem também os operadores unários, sendo eles o -, +, -- e ++. A seguir, a função de cada um:

  • -: Converte um operando em um número e o negativa. Por exemplo, se temos uma variável var com a string "10" como seu valor, escrevendo -var, teríamos o numero -10 como resultado. É o mesmo que utilizar a função Number() colocando o sinal de menos antes da função.
  • +: Converte um operando em um número. Por exemplo, se temos uma variável var com a string "10" como seu valor, escrevendo +var, teríamos o numero 10 como resultado. É o mesmo que utilizar a função Number().
  • --: Descresce o valor de um operando em uma unidade. Por exemplo, se temos uma variável var com o número 10 como seu valor, escrevendo var--, teríamos o numero 9 como resultado. É o mesmo que escrever var = var - 1 ou var -= 1.
  • ++: Acresce o valor de um operando em uma unidade. Por exemplo, se temos uma variável var com o número 10 como seu valor, escrevendo var++, teríamos o numero 11 como resultado. É o mesmo que escrever var = var + 1 ou var += 1.

É importante lembrar que, em operações aritméticas em expressões matemáticas, assim como na matemática usual, a ordem importa. Deste modo, operações aritméticas em Javascript seguem a mesma prioridade em expressões: parênteses, potências, multiplicações e divisões e adição e subtração.
