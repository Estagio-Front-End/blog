DOM em Javascript

JAVASCRIPT

O DOM é o modelo no qual o HTML de uma página é visto como uma árvore e cada um de seus elementos são suas ramificações. O navegador gera esta árvore ao ler o documento HTML e, por meio da conexão com a tag script, o Javascript pode interagir com esta árvore. Esta interação pode se dar por seleção de elementos diretamente ou pelas suas descendências/ascendências dentro da árvore, manipulação de elementos, de seus atributos ou de seu estilo, além de trabalho com eventos.

Seleção de elementos diretamente

Para a seleção de itens no document, que é o nome que se dá para a árvore do DOM, é possível utilizar vários métodos do elemento document, acessando-os com a notação de ponto.

  • document.getElementById(id): busca e retorna o elemento identificado com o atributo id passado como argumento. Só retorna um elemento pois cada id deve ser único dentro do documento HTML.
  • document.getElementsByName(name): busca e retorna uma NodeList com os elementos identificados com o atributo name passado como argumento. Uma NodeList é um elemento array-like, ou seja, parecido com um array, mas que não suporta os métodos de array.
  • document.getElementsByTagName(tag): busca e retorna uma HTMLCollection com os elementos da tag passada como argumento. A HTMLCollection, assim como a NodeList, é um objeto array-like.
  • document.getElementsByClassName(classe): busca e retorna uma HTMLCollection com os elementos que possuem como atributo class o valor passado como argumento.
  • document.querySelector(tag/id/class): busca e retorna o elemento com base no argumento que foi passado. Podem ser buscados elementos por tag, indicando a tag, por id, colocando um ponto (.) antes do valor do identificador e por classe, colocando um jogo da velha (#) antes do valor do atributo. Retorna apenas um elemento e, caso haja vários, retornará a primeira correspondência dentro da árvore.
  • document.querySelectorAll(tag/class): busca e retorna uma NodeList com os elementos identificados com base no argumento passado. É possível buscar por tag e por classe, assim como no método querySelector. Não faz sentido buscar por id, pois surtirá o mesmo efeito que o método anterior, visto que ids são únicas dentro do documento HTML.

Seleção de elementos pela ascendência/descendência

Com a seleção de um dado elemento pelos métodos citados anteriormente, é possível, por meio de algumas propriedades, acessar os elementos pais, elementos filhos ou elementos irmãos daquele que foi inicialmente selecionado. Estas propriedades são:

  • Selecionando a ascendência: para buscar elementos pais, utiliza-se as propriedades parentElement ou parentNode. Desta forma, o código document.getElementById("id").parentElement irá retornar o elemento pai do elemento com id "id" e todos os seus filhos, incluindo o elemento buscado originalmente.
  • Selecionando a descendência: para buscar elementos filhos, utilizamos a propriedade childNodes, que, por sua vez, retornará uma NodeList com todos os elementos filhos. É também possível utilizar a propriedade firstChild para acessar o primeiro elemento filho ou lastChild, para acessar o último.
  • Selecionando elementos no mesmo nível: para buscar elementos do mesmo nível, ou seja, elementos irmãos, é possível utilizar as propriedades previousSibiling para acessar o elemento irmão anterior e nextSibling para acessar o elemento irmão posterior.

Se é necessário buscar elementos não adjacentes, é possível subir e descer na árvore dos Nodes, como são chamados os elementos. Tomando como exemplo uma lista não ordenada (ul) com 4 itens (li), se já temos selecionado o terceiro item e queremos acessar o primeiro por meio dele, não é possível utilizar a seleção de elemento do mesmo nível. Desta forma, é possível subir na árvore (parentNode) e descer novamente para o elemento que se quer (firstChild).

Manipulação de elementos

É possível, com Javascript, criar e manipular elementos HTML com métodos do próprio documento. Os elementos criados constarão na árvore do DOM disposta no navegador, mas não existirão no documento HTML original, são apenas criados dinamicamente pela programação em Javascript. Os principais métodos de manipulação são:

  • document.createElement(tag): com este método, é possível criar um único elemento da tag passada no argumento.
  • elemento.appendChild(elemento2): com esse método é possível adicionar filhos ao elemento pai, ou seja, se criamos uma ul e uma li com o método createElement, é com o método appendChild que "inserimos" o item de lista como filho da lista não ordenada.
  • elemento.removeChild(elemento): com este método, é possível remover elementos do DOM. É importante frisar que, caso o elemento tenha filhos, os filhos também serão removidos com a utilização deste método.
  • elemento.textContent: este método insere conteúdo textual no elemento, muito utilizado para indicar o texto em parágrafos, âncoras ou itens de lista.

Manipulação de atributos

Uma parte importante dos elementos são seus atributos, principalmente em elementos como imagens, âncoras e inputs. É possível adicioná-los e manipulá-los a depender da necessidade com os seguintes métodos:

  • elemento.setAttribute(atributo, valor): este método cria o atributo indicado no primeiro argumento com o valor indicado no segundo argumento. Em uma imagem, por exemplo, podemos inserir seu texto alternativo com elemento.setAttribute('alt', 'Texto alternativo da imagem').
  • elemento.getAttribute(atributo): este método retorna o valor do atributo passado como argumento pertencente ao elemento no qual o método está sendo aplicado. Se o elemento não possuir o atributo, ele retorna null.
  • elemento.removeAttribute(atributo): este método remove o atributo passado como argumento pertencente ao elemento no qual o método está sendo aplicado. Caso o elemento não possua este atributo, ele retorna undefined.
  • elemento.hasAttribute(atributo): este método retorna um booleano, true caso o elemento possua o atributo passado como argumento, false, caso contrário.

Manipulação do estilo dos elementos

Apesar da estilização ser função do CSS, é possível aplicar e modificar estilos dos elementos por meio do Javascript e das propriedades style, className e classList.

  • proriedade style: esta propriedade dá acesso às propriedades do CSS, de forma que se pode modificar ou aplicar um estilo específico à um elemento. É importante frisar que deve-se utilizar camel case ao indicar as propriedades CSS, visto que não é possível utilizar a notação com hífen. A propriedade border-radius no CSS, por exemplo, fica elemento.style.borderRadius em Javascript, para a qual se pode atribuir um valor.
  • propriedade className: com esta propriedade, é possível retornar ou atribuir uma ou mais classes (separadas por espaço) ao elemento que se referencia. Caso seja atribuída à uma variável, irá retornar uma string com a(s) classe(s) do elemento. Caso um valor do tipo string seja atribuído a ela, esta será a nova (ou novas) classe do elemento.
  • propriedade classList: esta propriedade dá acesso à lista de classes individualmente, permitindo adição, remoção e manipulação das classes. Com o método add(classe), é possível adicionar uma classe ao elemento; com o método remove(classe) é possível remover uma classe do elemento; com o método replace(classe atual, classe nova) é possível substituir uma classe por outra e, com o método toggle(classe), é possível alternar uma classe: caso exista, será removida; caso não exista, será adicionada.
