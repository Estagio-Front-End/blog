Arrays em Javascript

JAVASCRIPT

Arrays são um tipo de dado não primitivo que se assemelha à listas. Estas listas contém métodos herdados que realizam operações de busca, edição e criação de novos arays. Para a inicialização de um array, pode-se proceder de duas formas: criando um array literal ou por meio de uma função construtora.

Com a notação literal, podemos criar um array vazio utilizando apenas colchetes ([]) ou criar um array populado, indicando os dados a serem listados dentro do par de colchetes.

Com a função construtora, podemos criar um array vazio, não passando nenhum argumento no objeto Array(), podemos criar um array com comprimento definido, passando um parâmetro no objeto, sendo este o comprimento e podemos criar um array populado, passando os dados a serem listados como parâmetros no objeto Array, separados por vírgulas.

Para acessar elementos de arrays, é necessário utilizar uma notação de colchetes e, dentro deles, o índice do elemento que se deseja acessar. É importante lembrar que, em arrays, o índice se inicia em zero. Desta forma, um array com 5 posições possui os índices de 0 a 4. Para acessar, por exemplo, o segundo elemento de um array chamado arr, temos que utilizar o código arr[1].

Uma propriedade muito importante para acessar elementos de arrays, entre outras funções, é o comprimento de um array. A propriedade lenght pode ser útil em situações em que não se sabe quantos elementos o array tem e se deseja, por exemplo, acessar seu último elemento. Para tal, utilizamos o código arr[arr.length - 1], visto que, como o índice inicia em 0, um array de n elementos terminará no índice (n - 1). Outra função importante do length é definir o ponto de parada de um loop que deve percorrer todo o array. Esta propriedade pode ser reatribuída em valor, em que, quando o valor é maior que o array existente, o array irá crescer em seu comprimento e, caso o valor seja menor, elementos finais do array serão excluídos.

Métodos em arrays

Tão importante quanto a criação de arrays é a manipulação destes para as mais diversas finalidades dentro de um código. Os arrays possuem métodos de busca, edição e criação de novos arrays. São eles:

Métodos de busca no array

  • find(função de teste): este método busca e retorna o primeiro elemento dentro do array que satisfaça a condição imposta na função passada como argumento. Caso nenhum elemento satisfaça a condição, o retorno será undefined.
  • findIndex(função de teste): este método busca e retorna o índice do primeiro elemento dentro do array que satisfaça a condição imposta na função passada como argumento. Caso nenhum elemento satisfaça a condição, o retorno será -1.
  • indexOf(alvo): este método retorna o índice do elemento indicado como argumento, caso ele exista dentro do array. Caso ele não exista, o retorno é -1.
  • some(função de teste): este método indica se algum dos elementos do array atende à condição imposta na função de teste passada como argumento. Caso positivo, o método retornará true, caso negativo, false.
  • every(função de teste): este método indica se todos os elementos do array atendem à condição imposta na função de teste passada como argumento. Caso positivo, o método retornará true, caso negativo, false.
  • includes(alvo): este método retornará true ou false a depender de se o elemento passado como argumento do método exista dentro do array ou não.

Métodos de edição do array

  • forEach(função): este método itera por todos os elementos do array e executa a função passada como argumento para cada um.
  • push(elemento): este método adiciona o(s) elemento(s) passado(s) como argumento no final do array.
  • pop(): este método remove o último elemento do array e retorna este elemento (caso utilizado como valor de variável ou outros contextos como condicionais e funções).
  • shift(): este método remove o primeiro elemento do array e retorna este elemento (caso utilizado como valor de variável ou outros contextos como condicionais e funções).
  • unshift(elemento): este método adiciona o(s) elemento(s) passado(s) como argumento no início do array.
  • sort(função de comparação): este método ordena os elementos dentro de um array. Pode ou não receber uma função de comparação como argumento, em que, caso não receba, irá ordenar os elementos de acordo com seu código Unicode.
  • fill(alvo, posição inicial, posição final): este método preenche os elementos de um array com um elemento alvo indicado no argumento. As posições iniciais e finais, que são opcionais, delimitam os índices nos quais o método irá atuar, sendo a posição final não inclusiva. Desta forma, se inserirmos no código arr.fill(10, 2, 4) os elementos de índices 2 e 3 do array chamado arr serão preenchidos com o número 10.
  • reverse(): este método inverte a ordem dos elementos dentro de um array.
  • splice(índice inicial, quantidade elementos retirados, elemento novo): este método modifica os elementos de um array incluindo valores novos e deletando (ou não) valores antigos. Deste modo, inserindo no código arr.splice(1, 0, "Camila") o array chamado arr terá seu elemento de índice 1 trocado por "Camila", mas sem retirar elementos do array original.
  • reduce(função, valor inicial): este método itera por todo o array e reduz os elementos a um único valor seguindo uma função redutora, indicada como argumento. A função redutora, em geral, recebe 4 argumentos: o acumulador, que irá acumular o resultado da redução até o final; o valor atual e seu índice; e o valor inicial do acumulador. Ao final da iteração, o método retorna o acumulador, que terá o resultado final.

Métodos de criação de novo array

  • Array.from(elemento, função): uma outra forma de criação de arrays, gerando um array a partir de qualquer objeto iterável. Pode ou não receber uma função modificadora, em que os elementos primeiramente serão modificados para, depois, integrarem o array.
  • Array.of(elementos): uma outra forma de criação de arrays, gera um array a partir dos elementos indicados no argumento da função.
  • map(função modificadora): este método itera pelo array e, para cada elemento, aplica uma função modificadora indicada como argumento, criando um novo array com os elementos modificados.
  • filter(função afuniladora): este método cria um novo array com todos os elementos do array original que satisfaçam a condição indicada na função afuniladora.
  • join("divisória"): este método junta os elementos do array e os transforma em uma string, separados pela divisória indicada no argumento. Se nenhuma divisória for passada, os elementos sairão todos colados.
  • concat(outro array): este método concatena 2 arrays, sendo o array passado no argumento incluso no final do array original.
  • flat(): este método cria um novo array, concatenando quaisquer elementos arrays no array original. Por exemplo, inserindo no código [1,2,3,4,[5,6]].flat() teremos um novo array [1,2,3,4,5,6].
  • slice(posição inicial, posição final): este método recorta e retorna uma parte do array, indicada pelas posições inicial e final. A posição final, assim como no método fill(), não é inclusiva.
