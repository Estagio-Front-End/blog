CSS inline, interno e externo

CSS BÁSICO

O CSS é uma linguagem de estilo por meio da qual documentos em HTML ou XML tem sua apresentação para o usuário descrita. Existem 3 formas de adicionar a estilização à um documento HTML: de forma inline, interna ou externa.

No CSS inline, a estilização a ser aplicada é descrita dentro da própria tag HTML que será estilizada, dentro do documento HTML, por meio do atributo style.

No CSS interno, a estilização a ser aplicada é descrita dentro de uma tag style, posicionada entre as tags head e body do documento HTML. Para identificação das tags nas quais cada estilo será aplicado, são utilizados os seletores, classes ou ids utilizados no código HTML.

No CSS externo, a estilização a ser aplicada é descrita dentro de um documento com extensão .css, que fica à parte do documento HTML. Já que a descrição da estilização não fica inserida no documento HTML, é necessário que os documentos .html e .css sejam relacionados por meio de uma tag link, a ser inclusa dentro da tag head do documento HTML. Para identificação das tags a serem estilizadas, assim como no CSS interno, são utilizados seletores, classes e ids.

Atualmente, a melhor prática é utilizar o CSS externo, visto que mantém o código organizado para cada função (marcação e estilização) e permite a reutilização de código, em que uma folha de estilo (documento .css) pode ser aplicado para vários documentos HTML.
